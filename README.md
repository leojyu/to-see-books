# To-Read

To-Read 是一个单页 Web 应用程序，用于发布、浏览和阅读小说。它提供了用户认证、书籍管理、阅读器以及数据导入导出功能，所有数据均存储在 JSONBin 云端，支持多用户独立账号。

## 功能特性

- 用户系统：注册、登录、登出，密码使用 SHA-256 加密存储。
- 书籍发布：登录用户可上传 `.txt` 文件作为小说正文，填写书名、作者和简介，发布后所有用户可见。
- 发现与阅读：在“发现”页浏览所有已发布的书籍，点击任意书籍进入阅读器，支持全屏模式。
- 最近阅读：自动记录最近打开的书籍，方便快速继续阅读。
- 数据持久化：所有书籍和用户数据存储在 JSONBin 云端，支持离线缓存（本地 localStorage 作为降级）。
- 数据导入/导出：支持将全部数据（书籍+用户）导出为 JSON 文件，也支持导入之前导出的数据文件，方便备份与迁移。
- 草稿保存：发布表单中的内容会自动保存到本地，避免意外丢失。
- 自适应界面：针对移动设备进行了响应式优化，在手机和电脑上均有良好体验。

## 技术栈

- 纯原生：HTML5 + CSS3 + Vanilla JavaScript（无第三方库依赖）。
- 加密：Web Crypto API（SHA-256）用于密码哈希。
- 存储后端：JSONBin（通过 REST API 进行数据读写）。
- 本地存储：localStorage 用于缓存云端数据、保存用户登录状态、最近阅读列表和草稿。
